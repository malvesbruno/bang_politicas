Política de Uso de Dados – Bang!

Esta Política de Uso de Dados descreve como o Bang! coleta, utiliza e protege as informações dos usuários. Nosso compromisso é garantir transparência e segurança em relação aos seus dados.

Coleta de Dados

O Bang! coleta apenas informações necessárias para:

  • Permitir a jogabilidade online (ex.: criação de perfil, partidas e interações entre jogadores).
  • Salvar informações de progresso localmente no dispositivo (ex.: estatísticas, recompensas e preferências de jogo).

Não coletamos dados pessoais sensíveis, como documentos, localização precisa ou informações financeiras.

Uso dos Dados

Os dados coletados são utilizados exclusivamente para:

  • Garantir o funcionamento correto do jogo.
  • Salvar o progresso do usuário.
  • Possibilitar partidas online contra outros jogadores.

Não utilizamos os dados para publicidade direcionada ou venda a terceiros.

Compartilhamento de Dados

Não compartilhamos nem vendemos suas informações a empresas, organizações ou indivíduos externos. Todos os dados são usados apenas dentro do Bang!.

Armazenamento dos Dados

  • Dados locais: ficam armazenados diretamente no seu dispositivo e podem ser apagados a qualquer momento, desinstalando o aplicativo
  • Dados online: são mantidos apenas para possibilitar o funcionamento das partidas e rankings, sendo utilizados de forma restrita e segura.

Segurança

Adotamos medidas técnicas para proteger seus dados contra acesso não autorizado, alteração, divulgação ou destruição.

Direitos do Usuário

Você pode:

  • Solicitar a exclusão de seus dados relacionados à conta online.
  • Desinstalar o aplicativo para remover todos os dados locais

Alterações nesta Política

Podemos atualizar esta Política de Uso de Dados periodicamente. Caso haja mudanças significativas, notificaremos os usuários dentro do próprio aplicativo
